# Rock, Paper, Scissors Game

A simple browser-based Rock, Paper, Scissors game built with **HTML**, **CSS**, and **JavaScript**.

## Features

- Interactive user interface
- Play against the computer
- Real-time score updates and instant results
- Responsive design
- Themed images for rock, paper, and scissors
- Beginner-friendly code for easy learning or customization

## How to Play

1. Open `game.html` in your web browser.
2. Choose rock, paper, or scissors to play against the computer.
3. View instant feedback and live scores.

## Files

- `game.html` — Main HTML file
- `game.css` — Stylesheet
- `game.js` — Game logic
- `rock.png`, `paper.png`, `scissors.png` — Game images

Feel free to use, modify, or improve this project for your own learning or enhancements!
